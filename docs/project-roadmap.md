# Project Roadmap

## Completed MVP

- Laravel 12 backend
- Vue 3 frontend
- Email/password registration and login
- Google OAuth
- Sanctum bearer-token authentication
- User profile management
- Project CRUD
- Task CRUD inside projects
- Dashboard statistics
- Global project/task search
- Backend feature tests
- Vercel/Railway deployment documentation

## Near-Term Improvements

- Password reset flow
- Email verification
- Frontend form refinements
- Better loading and toast feedback
- Task filters by status, priority, and due date
- Project archive state
- CI checks for `php artisan test` and `npm run build`

## Production Hardening

- Move production data from SQLite to PostgreSQL
- Add database backups
- Add centralized logging
- Add rate limiting for auth endpoints
- Add monitoring and uptime checks

## Future Product Features

- Team workspaces
- Project members and roles
- Kanban board
- Comments
- File attachments
- Notifications
- Activity log
- Calendar view
- Due-date reminders
